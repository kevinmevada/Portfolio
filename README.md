# Kevin Mevada - AI Engineer Portfolio

A modern, responsive portfolio website showcasing AI/ML projects, experience, and technical expertise with interactive 3D gyroscope visualization.

## 🚀 Features

- **Physics-Based 3D Gyroscope**: Real-time simulation using Euler's equations for rigid body dynamics
- **Responsive Design**: Mobile-first approach with smooth animations and interactions
- **Modern UI/UX**: Professional aesthetics with custom cursor and smooth scrolling
- **Performance Optimized**: Efficient rendering with optimized animations
- **Production Ready**: Clean, modular OOP architecture

## 📁 Project Structure

```
Portfolio/
├── index.html              # Main portfolio page
├── css/
│   └── styles.css          # All styling and animations
├── js/
│   ├── gyroscope.js        # 3D gyroscope engine
│   ├── portfolio-ui.js     # UI interactions and components
│   └── portfolio-data.js   # Centralized data management
├── assets/                 # Images, fonts, and media
├── docs/                  # Documentation and guides
└── README.md              # This file
```

## 🛠️ Technology Stack

### Frontend
- **HTML5**: Semantic markup with accessibility in mind
- **CSS3**: Modern features with CSS variables and animations
- **JavaScript ES6+**: OOP concepts with modular architecture
- **Three.js r128**: 3D graphics and physics simulation

### Design System
- **Custom Properties**: CSS variables for consistent theming
- **Responsive Grid**: Mobile-first layout system
- **Professional Typography**: Syne + JetBrains Mono font combination
- **Interactive Elements**: Hover states, transitions, and micro-interactions

## 🎯 Key Components

### GyroscopeEngine Class
- **Physics Simulation**: Euler's equations for realistic gyroscope motion
- **3D Rendering**: Three.js scene management and optimization
- **Interactive Controls**: Mouse-responsive torque application
- **Performance**: Efficient animation loops and memory management

### PortfolioUI Class
- **Custom Cursor**: Interactive cursor system with visual feedback
- **Smooth Scrolling**: Enhanced navigation experience
- **Animation System**: Intersection Observer for scroll animations
- **Status Management**: Dynamic status indicators

### PortfolioData Class
- **Centralized Data**: Single source of truth for all content
- **Validation**: Data integrity and type checking
- **Export/Import**: JSON-based data management
- **Modular Structure**: Easy content updates and maintenance
